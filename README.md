# School Library
> This applications has the following functionality:
- Add new students or teachers.
    
- Add new books.
    
- Save records of who borrowed a given book and when.


## Built With 🔨
- Ruby

## Testing 
- Run `Bundle install`
- To run test write `rspec ./spec` on the console
